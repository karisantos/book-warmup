## Exercise 2

Write a look to calculate the sum of 1, 2, 3, ...., 10.

{% exercise %}
Calculate the sum of 1, 2, 3, ...., 10
{% initial %}
var a = 1
var b = 10
var sum = 0
{% solution %}
var a = 1
var b = 10
var sum = 0
var i;
for (i=a;i<=b;i++){
	sum +=i;
}
{% validation %}
assert(sum == 55)
{% endexercise %}
